# World Cup Signal Style Guide

## Color Palette

Use the screenshot palette as the source of truth for new pages.

| Token | Hex | Role |
| --- | --- | --- |
| `--purple-illusion` | `#BDB5FB` | Soft highlights, pale panels, hero eyebrow text, subtle glows |
| `--lightish-purple` | `#A45CE4` | Primary calls to action, value tags, confident accent moments |
| `--dragon-purple` | `#5E42BE` | Active filters, priority pills, strong section emphasis |
| `--impression-obscurity` | `#151B7D` | Deep headers, table headers, secondary dark accents |
| `--velvet-vortex` | `#500A68` | High-intensity notes, hot tags, risk and value emphasis |
| `--ultramarine-shadow` | `#080441` | Hero overlay, footer, deepest background color |

## Theme Rules

- Keep the page mostly light with `#F7F5FF` and white cards so the purple system feels sharp rather than heavy.
- Use `--ultramarine-shadow` and `--impression-obscurity` for football-night drama: hero overlays, footers, table headers, and dense information blocks.
- Use `--lightish-purple` for the main action on each screen. Avoid using it on every badge; it should feel important.
- Use `--purple-illusion` for soft support surfaces, dividers, and calm background bands.
- Use `--velvet-vortex` sparingly for alerts, hot picks, and betting/risk notes.

## Component Direction

- Cards stay white, 8px radius, with lavender borders and a soft navy shadow.
- Buttons and filter states should use filled purple backgrounds with white text.
- Data-heavy sections should alternate between white pages and pale lavender bands.
- Progress bars can use the full palette as a gradient: velvet to lightish purple to purple illusion.
- Keep typography compact and dashboard-like: Inter, bold section labels, and short headings.

## Betting Labels

- Use **Safer** for lower-payout picks that are more likely to land.
- Use **Value** when the price looks attractive for the risk.
- Use **Goals** for over/under, team-goals, or both-teams-to-score ideas.
- Use **Spicy** for higher-risk, higher-payout ideas that should be smaller-stake.
- Use **Avoid** when the outcome may happen but the odds or uncertainty are not worth it.
